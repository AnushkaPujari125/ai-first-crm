AI-First CRM HCP Module

An AI-powered CRM module for Healthcare Professional (HCP) interaction logging built using React, Redux, FastAPI, Groq LLM, SQLite, and LangGraph concepts.

Features
AI-powered interaction logging
Conversational chat interface
Auto form filling using AI extraction
HCP interaction management
SQLite database storage
Redux state management
FastAPI backend APIs
Responsive modern UI

AI Features

The AI assistant can:

Extract HCP Name
Detect Interaction Type
Extract Sentiment
Detect Topics Discussed
Detect Materials Shared
Save Interaction Records
Show All Records
Delete Records
Edit Existing Records
LangGraph Tools Used
1. Log Interaction Tool

Captures interaction details from natural language and stores them in database.

2. Edit Interaction Tool

Allows modification of existing interaction records through AI instructions.

3. Delete Interaction Tool

Deletes records using conversational commands.

4. Show Records Tool

Displays all stored HCP interactions.

5. Summarization Tool

Uses LLM to summarize conversational interaction data.
